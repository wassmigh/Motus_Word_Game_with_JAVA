# 🎯 Motus Word Game (Java)

**Motus** est un jeu de devinettes textuel en Java, inspiré des puzzles classiques. Le joueur doit deviner un mot secret en un nombre limité d’essais.

Le projet propose deux modes :
- **Mode Jeu** : pour deviner le mot mystère.
- **Mode Configuration** : pour gérer le dictionnaire de mots.

Le jeu intègre également un système de statistiques pour suivre les performances du joueur.

## 🧠 Points Clés

- Jeu entièrement textuel en Java
- Dictionnaire personnalisable
- Suivi des victoires, défaites et tentatives restantes
- Interface simple via la console

🎮 *Un projet simple, ludique et formateur pour pratiquer la programmation orientée objet en Java !*
